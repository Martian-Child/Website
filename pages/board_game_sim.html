Board Game Simulator

About

This is a game that is meant to be a framework for creating physics based board game simulations that are great to play with a group of friends or family! It features a deck that shuffles, and can be drawn from, 3D cursor, a magnifying glass, picking up objects, interaction sounds and impact sounds for different objects via scriptable objs, an interactive rule book, an interactive notebook, an interactive radio that can be tuned to different songs, generic object interaction so differenct objects can be interacted with in different ways (for example: left click to on dice to roll dice, left click on card to flip it, and left click on deck to shuffle), and more!
